### 2. GitHub 명령어

#### **1. git status**

  : 현재 local repository 내에 변경 사항이 있는지 확인

#### **2. git add**

  : 현재 변경 사항에 대해 저장소에 저장

#### **3. git commit -m "msg"**

  : git add 한 내용에 대한 이력을 저장

#### **4. git push**

  : local repository(ex. PC)에서 Remote repository 로 저장

#### **5. git log**

  : 수행 된 내용들에 대한 history 출력
